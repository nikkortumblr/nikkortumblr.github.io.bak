Usage: http://api.barthelet.eu/backup/tumblr/YOUR-TUMBLR-USERNAME
Example: http://api.barthelet.eu/backup/tumblr/demo
You can add a count argument to retrieve as many posts as you want (use count=all to retrieve them all):
http://api.barthelet.eu/backup/tumblr/YOUR-TUMBLR-USERNAME/?count=NUMBER-OF-POSTS
